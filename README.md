# @reactiff/app-data-client

Simple, code-first, virtual API client for your app

- simple, efficient CRUD operations 
- code-first to configure your app's Entity types and Fields 
- use it anywhere, even in [codepen](http://codepen.io)

[![NPM](https://img.shields.io/npm/v/@reactiff/app-data-client.svg)](https://www.npmjs.com/package/@reactiff/app-data-client) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

<br>

## Install

```bash
yarn add @reactiff/app-data-client
```

## Usage
```tsx
import AppDataClient from '@reactiff/app-data-client'

const api = new AppDataClient();

// do stuff with it

```
